## UGC Ad Prompt Engineering Project

### Objective
To create a reusable AI prompt system that generates high-converting UGC-style ad content.

### Business Chosen
Online skincare brand (D2C).

### What This Repository Contains
- Structured AI prompts for UGC ads
- Multiple hooks for ad testing
- Full UGC ad scripts
- Conversion-focused CTAs

### Platforms
- Instagram Reels
- Short-form paid ads

### Tools Used
- ChatGPT
- AI video tools
- CapCut

### Use Case
This system can be used by:
- Marketing agencies
- D2C brands
- Freelancers creating UGC ads
